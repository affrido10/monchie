# Lumina: Philosophical Minds & Language Notes

A minimalist personal workspace for capturing philosophical reflections and linguistic explorations.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set `VITE_SUPABASE_ANON_KEY` in [.env.local](.env.local)
3. Run the app:
   `npm run dev`
